群晖NAS硬盘已损毁怎么提取数据

Posted on 2025-06-06
  1. 将硬盘拔下来插到台式机(windows)上
  2. 下载安装UFS explorer professional recovery（需注册或破解）
  3. 找到对应文件夹右键选取”copy with pararmeters…”，选取目的文件夹
  4. 等待拷贝完成。
  5. 读取Save Report查看有哪些文件拷贝存在异常。

使用Github建个人博客

Posted on 2025-06-06

步骤：

本地电脑安装

  1. 安装Node

  2. 安装yarn

  3. 安装hexo

  4. 新建一个Blog项目

  5. 启动本地Hexo服务，查看测试页面

    本机测试页面：

    http://localhost:4000/

网站修改

  1. 安装Next主题（或其它主题）

  1. 修改根文件夹下的_config.yml文件，theme指向next

  2. hexo server启动服务查看测试页面。

  3. 配置主题，以及网站_config.yml

相关命令

关联Github

  1. 新建一个Reposity ,名字.github.io

  2. 根目录下_config.yml填写Reposity信息

  3. deploy

  4. 弹出填写用户名密码（github帐号）

    username:

    password: 密码不是登陆密码，在github setting上找personal access token

    Developer settings

  5. deploy完成后，使用.github.io访问网站。

　

其它

增加阅读人数统计功能

  • 注册leancloud国际版

  • 创建应用

  • 创建class

  • 转到应用凭证界面

  • 修改theme文件夹下的_config.yml文件

换系统或者换电脑需要重装 hexo
 的时候，只需要在hexo
环境搭建好的基础上，只需要把 source
、themes
 、 _config.yml
 覆盖到新的博客目录下即可
